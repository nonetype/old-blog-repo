## bp hit시마다 명령 실행
```c
(gdb) b do_mmap_pgoff

(gdb) command 1

End with a line saying just "end".
>print addr
>print len
>print prot
>end
(gdb)
```

## gdb에서 malloc으로 Heap 포인터 받아오기(malloc, kmalloc, vmalloc)
```c
(gdb) p vmalloc(32)
$4 = (void *) 0xffffc90000372000
(gdb) set {char[25]}$4="thisisaverylongstring\n"
(gdb) x/s $4
0xffffc90000372000:	"thisisaverylongstring\n"
```
